# Task-Manager
📝 Task Manager (CRUD)
A clean and minimal web-based task manager designed to help you stay organized and productive. This project demonstrates the core principles of CRUD (Create, Read, Update, Delete) operations through a simple yet elegant interface.
✨ Overview
This task manager allows users to quickly add, view, and delete tasks in real time. Built with simplicity in mind, it focuses on delivering a smooth user experience without unnecessary complexity.
🚀 Features
➕ Add tasks using an intuitive input bar
📋 Instantly display tasks in a structured list
❌ Delete tasks with a single click
⚡ Real-time updates without page reload
🎯 Minimal and distraction-free design
🛠️ Tech Stack
HTML – Structure
CSS – Styling and layout
JavaScript – Functionality and dynamic updates
💡 How It Works
Enter your task in the input field
Press enter or click the add button
View your tasks appear instantly
Delete tasks once completed or no longer needed
📌 Purpose
This project was built to strengthen understanding of fundamental web development concepts, especially CRUD operations, while maintaining a clean and user-friendly design.
